# Magnet Core — Magnetism & Electromagnetic Physics

**Difficulty:** 5 | **Theme:** Dark industrial reds, ember particles | **Final map — portal returns to Atom Meadows after completion**

---

## Learning Objectives

- Understand magnetic poles: opposite poles attract, like poles repel
- Learn that magnetic fields are invisible but mappable using iron filings
- Understand electromagnetism: electric current through a coil creates a magnetic field
- Learn that Earth itself is a giant magnet with magnetic north and south
- Understand projectile motion and how magnetic forces can accelerate objects
- Learn real applications: maglev trains, railguns, magnetic shielding, compasses

## Trajectory Puzzles

Each puzzle requires launching a projectile to hit a target, with increasing complexity:
- **Puzzle 1:** Angle + velocity only (no magnets) — introduces the mechanic
- **Puzzle 2:** Static magnetic fields affect the path
- **Puzzle 3:** Moving targets + field navigation
- **Puzzle 4:** Wind + magnets + moving targets — final challenge

The Compass (equipped item) reveals magnetic field lines in navigation puzzles.
