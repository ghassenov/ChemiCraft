# Recycling Fields — Materials Science & Circular Economy

**Difficulty:** 2 | **Theme:** Industrial browns and yellows, falling leaves | **Portal unlocks after:** completing all 4 quests

---

## Learning Objectives

- Identify common material types: plastic (PET, HDPE, PVC, PP, PS), glass, metal, paper, organic
- Understand waste sorting: which bin each material belongs in
- Learn that recycling is physical processes (melting, crushing, pulping, sorting) — not chemical
- Understand the circular economy: waste is a resource in the wrong place
- Learn real-world recycling statistics: glass is infinitely recyclable, aluminum saves 95% energy, only 9% of plastic has been recycled globally
- Understand that contamination ruins entire recycling batches

## Sorting Minigame

Items scroll on a conveyor belt — drag each to the correct bin:
- **Yellow** = Plastic, **Green** = Glass, **Grey** = Metal, **Blue** = Paper, **Brown** = Organic

Wrong bin = contamination penalty. One wrong item can ruin a batch — just like real recycling.
