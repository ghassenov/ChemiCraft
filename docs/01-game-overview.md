# ChemiCraft — Game Overview

## Core Concept

ChemiCraft is a **web-based educational role-playing game (RPG)** that teaches chemistry through engaging quests, exploration, and hands-on experimentation. The player controls an avatar in a 2D tile-based map, interacts with NPCs who present chemistry-related problems (e.g., synthesizing air molecules to improve village ventilation), and uses a virtual laboratory to craft solutions.

Each completed quest rewards the player with coins, skill points, and inventory items, unlocking progressively harder challenges across multiple themed maps.

---

## Target Audience

- **Age range:** 10–16 years old
- **Prerequisites:** Basic reading skills
- **Difficulty curve:** Starts with simple atomic structure → advances to complex chemical reactions and precipitation

---

## Learning Objectives

1. **Understand atoms, molecules, and chemical bonds**
   - Identify elements, their symbols, and how they combine

2. **Learn chemical reactions**
   - Synthesis, decomposition, single/double displacement, precipitation, acid-base neutralization

3. **Apply theoretical knowledge in a virtual lab**
   - Combine reagents to produce target molecules and solve real-world (in-game) problems

4. **Develop systematic experimentation and critical thinking**
   - Trial-and-error in the lab, following clues from NPCs and the library

---

## Game Flow Summary

```
Login/Signup
    ↓
Select/Spawn on Map
    ↓
Explore (move avatar, discover POIs)
    ↓
Talk to NPC → Receive Quest
    ↓
Visit Library (learn theory / get hints)
    ↓
Go to Lab (mix reagents, craft solution)
    ↓
Return to NPC → Deliver Solution → Receive Reward
    ↓
Complete all quests on map → Unlock next map
```

## Maps & Progression

| Map # | Name                  | Topic                    | Example Quest                     |
|-------|-----------------------|--------------------------|-----------------------------------|
| 1     | Atom Meadows          | Atoms & Molecules        | Build H₂O, CO₂, N₂               |
| 2     | Reaction Ravine       | Simple Reactions         | Combustion of methane             |
| 3     | Precipitation Plateau | Precipitation Reactions  | Make lead iodide (yellow ppt)    |
| 4     | Acid-Base Atoll       | Acids & Bases            | Neutralize a toxic spill          |

Each map has its own **Lab**, **Library**, and **Shop** with unique available reagents and items.
